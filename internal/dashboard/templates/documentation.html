Documentation

quick reference for Mole pull request commands

commands
Standard review
lighter review for pull requests
/mole review

Runs the standard Mole review for the current pull request.

Deep review
full review with diagrams
/mole deep-review

Runs the deep Mole review for the current pull request, including diagrams.

Dig review
experimental
contextual review with repository exploration
/mole dig

Clona o repositório, explora o código com Sonnet (tool use), e revisa com Opus usando o contexto coletado.

Ignore pull request
skip future Mole reviews on this PR
/mole ignore

Stops Mole from reviewing this pull request again in the future.

feedback
Reactions
validate inline review comments

React to Mole inline comments to confirm issues or mark false positives.

👍
Confirm issue
Use when Mole correctly identified a real issue.
👎
False positive
Use when Mole flagged something that should not count as an issue.
